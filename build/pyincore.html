3. pyIncore¶

Please read through the instructions at least once completely before actually following them to avoid any installation problems!

pyIncore uses the Geospatial Data Abstraction Library (GDAL), which will be installed with pyincore package. The GDAL installation is global on Windows and Linux, even if you use virtual environments.
